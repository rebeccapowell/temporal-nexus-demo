# Skills Registry

This repository defaults to using `.agents/skills` for local skills.

Configured upstream sources:
- https://github.com/temporalio/skill-temporal-developer
- https://github.com/microsoft/aspire-skills
- https://github.com/github/awesome-copilot
- https://github.com/bradygaster/squad
- https://github.com/obra/superpowers

The Copilot setup workflow installs and refreshes these sources for cloud-agent sessions.

Resource index: https://awesome-copilot.github.io/
